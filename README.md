Parqueaderos Inteligentes

Sistema inteligente para la gestión y control de parqueaderos, orientado a mejorar la automatización, seguridad y eficiencia en la administración de espacios vehiculares.

Descripción General

Este proyecto implementa una solución integral para la administración de parqueaderos utilizando tecnologías modernas. Incluye módulos para el control de acceso, registro de vehículos, monitoreo de espacios disponibles, generación de reportes y paneles administrativos.

Características Principales

Registro y autenticación de usuarios

Gestión de entradas y salidas de vehículos

Control automático de disponibilidad

Dashboard administrativo

Reportes de ocupación

Integración con sensores (si aplica en la tesis)

Arquitectura escalable y modular

Tecnologías Utilizadas

Frontend:

HTML, CSS, JavaScript

Framework sugerido: React, Vue o interfaz nativa con JS (adaptable según tu tesis)

Backend:

Node.js o Java (Spring Boot), según tu implementación

API REST

JWT para autenticación

Base de Datos:

MySQL o PostgreSQL

ORM sugerido: Sequelize, Prisma o JPA (dependiendo del stack)

Cómo Ejecutar el Proyecto en Local
1. Clonar Repositorio

2. Backend
cd backend
npm install
npm run dev

3. Frontend
cd frontend
npm install
npm run dev
